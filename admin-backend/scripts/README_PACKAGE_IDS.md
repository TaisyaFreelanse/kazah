# Как получить ID пакетов

## Способ 1: Через скрипт (рекомендуется)

Запустите скрипт из папки `admin-backend`:

```bash
npm run get-package-ids
```

Или напрямую:

```bash
node scripts/get-package-ids.js
```

Скрипт выведет:
- Список всех пакетов с их ID, названиями и статусом
- ID всех пакетов (через запятую)
- ID только активных пакетов (через запятую)
- ID пакетов в формате JSON массива

## Способ 2: Через API (требует авторизации)

### Получить ID всех пакетов (требует токен администратора):

```bash
GET /api/packages/ids
Authorization: Bearer <ваш_токен>
```

Ответ:
```json
[
  {
    "id": 1,
    "name": "История Казахстана",
    "nameKZ": "Қазақстан тарихы",
    "nameRU": "История Казахстана",
    "isActive": true
  },
  ...
]
```

## Способ 3: Через публичный API (без авторизации)

### Получить ID только активных пакетов:

```bash
GET /api/public/packages/ids
```

Ответ:
```json
[1, 2, 3, ...]
```

## Способ 4: Через полный список пакетов

### Получить все пакеты с полной информацией:

```bash
GET /api/packages
Authorization: Bearer <ваш_токен>
```

или для активных пакетов:

```bash
GET /api/public/packages
```

В ответе каждый пакет содержит поле `id`.
